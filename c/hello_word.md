# Hola Mundo

Generar un archivo llamado `main.c` y agregar el siguiente texto:

```c
#include <stdio.h>

int main() {
  printf("Hola mundo");
  return 0;
}
```

Para compilar y ejecutar es con los siguientes comando:

```bash
gcc main.c -o programa && ./programa
```
